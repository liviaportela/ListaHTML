• Uso do Iframe em HTML

    O elemento <iframe> em HTML é usado para exibir uma página da web dentro de outra página da web.

    Sintaxe do Iframe

    O elemento <iframe> especifica um frame (quadro) inline. Um frame inline é usado para incorporar outro documento dentro do documento HTML atual.

    A sintaxe básica é:

                        <iframe src="url" title="descrição"></iframe>

    Dica: É uma boa prática sempre incluir o atributo title no <iframe>. Isso é usado por leitores de tela para ler o conteúdo do iframe.

    Iframe - Definir Altura e Largura

    Use os atributos height (altura) e width (largura) para especificar o tamanho do iframe. Por padrão, a altura e a largura são especificadas em pixels:

    Exemplo de Altura e Largura do Iframe

                        <iframe src="id.html" height="200" width="300" title="Exemplo de Iframe"></iframe>

    Resultado

    Ou você pode adicionar o atributo style e usar as propriedades CSS height e width: 

                        <iframe src="https://www.example.com" style="height: 200px; width: 300px;" title="Exemplo de Iframe"></iframe>

    Resultado

    Iframe - Remover a Borda

    Por padrão, um iframe tem uma borda ao redor. Para remover a borda, adicione o atributo style e use a propriedade CSS border:

    Exemplo de Iframe sem Borda

    Com CSS, você também pode mudar o tamanho, estilo e cor da borda do iframe:

                        <iframe src="basic.html" style="border: none;" title="Exemplo de Iframe"></iframe>

    Resultado

    Exemplo de Iframe com Borda Personalizada

                        <iframe src="attributes.html" style="border: 2px solid red;" title="Exemplo de Iframe"></iframe>

    Resultado

    Iframe - Destino para um Link

    Um iframe pode ser usado como destino para um link. O atributo target do link deve se referir ao atributo name do iframe:

    Exemplo de Iframe como Destino para um Link

                        <iframe src="emojis.html" name="iframe_a" title="Exemplo de Iframe"></iframe>
     <p>Visite <a href="/ListaHTML/src/pages/symbols.html" target="iframe_a">Símbolos HTML</a></p>

    Resultado

    Visite Símbolos HTML
