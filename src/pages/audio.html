• HTML Áudio

    O elemento HTML <audio> é usado para reproduzir um arquivo de áudio em uma página da web.

    Para reproduzir um arquivo de áudio em HTML, use o elemento <audio>:

    Exemplo

                        <audio controls>
        <source src="../media/igotafeeling.mp3" type="audio/mpeg">
        Seu navegador não suporta o elemento de áudio.
     </audio>

    resultado

    HTML Audio - Como Funciona

    O atributo controls adiciona controles de áudio, como play, pause e volume.

    O elemento <source> permite especificar arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

    O texto entre as tags <audio> e </audio> será exibido apenas em navegadores que não suportam o elemento <audio>.

    HTML <audio> Autoplay

    Para iniciar um arquivo de áudio automaticamente, use o atributo autoplay:

    Exemplo

                        <audio controls autoplay>
        <source src="../media/igotafeeling.mp3" type="audio/mpeg">
        Seu navegador não suporta o elemento de áudio.
     </audio>

    Resultado

    Nota: Os navegadores geralmente não permitem autoplay na maioria dos casos. No entanto, autoplay com mudo é sempre permitido.

    Tipos de Mídia para Áudio HTML

    Formato do Arquivo Tipo de Mídia
    MP3 audio/mpeg
    OGG audio/ogg
    WAV audio/wav

    HTML Audio - Métodos, Propriedades e Eventos

    O HTML DOM define métodos, propriedades e eventos para o elemento <audio>.

    Isso permite carregar, reproduzir e pausar áudios, bem como definir duração e volume.

    Há também eventos DOM que podem notificar você quando um áudio começa a tocar, é pausado, etc.

    Tags de Áudio HTML

    Tag Descrição
    <audio> Define conteúdo de som
    <source> Define múltiplos recursos de mídia para elementos de mídia, como <video> e <audio>.
