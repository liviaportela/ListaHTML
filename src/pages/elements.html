• Elementos HTML

    Um elemento HTML é tudo aquilo entre a tag de abertura e a tag de fechamento:

                        <nome_da_tag>Conteúdo aqui...</nome_da_tag>

    Exemplo de alguns elementos HTML:

                        <h1>Meu primeiro título.</h1>
     <p>Meu primeiro parágrafo.</p>
    Tag inicial Conteúdo do elemento Tag final
    <h1> Meu primeiro título </h1>
    <p> Meu primeiro parágrafo <p>
    <br> none none

    Elementos HTML Aninhados

    Os elementos HTML podem ser aninhados (o que significa que elementos podem conter outros elementos).

    Todos os documentos HTML consistem em elementos HTML aninhados.

    O exemplo a seguir contém quatro elementos HTML (<html>, <body>, <h1>, e <p>):

    Exemplo

                    <!DOCTYPE html>
        <html>
        <body>
            <h1>Meu primeiro título</h1>
            <p>Meu primeiro parágrafo.</p>
        </body>
        </html>

    Resultado

    Meu primeiro título

    Meu primeiro parágrafo.

    Explicação do Exemplo

    O elemento <html> É o elemento raiz e define todo o documento HTML.

    Ele tem uma tag de abertura <html> e uma tag de fechamento </html>.

    Dentro do elemento <html> há um elemento <body>, que define o corpo do documento. 

                        <body>
        <h1>Meu primeiro título</h1>
        <p>Meu primeiro parágrafo.</p>
    </body>

    O elemento <body> define o corpo do documento.

    Possui uma tag inicial <body> e uma tag final </body>

    Dentro do elemento <body>, há dois outros elementos: <h1> e <p>. 

                        <h1>Meu primeiro título</h1>
     <p>Meu primeiro parágrafo.</p>

    Nunca Pule a Tag Final

    Alguns elementos HTML serão exibidos corretamente, mesmo se você esquecer a tag final:

    Exemplo

                        <html>
        <body>
            <p>Isso é um parágrafo
            <p>Isso é um parágrafo
        </body>
    </html>

    Resultado

    Isso é um parágrafo

    Isso é um parágrafo

    No entanto, nunca confie nisso! Resultados inesperados e erros podem ocorrer se você esquecer a tag final!

    Elemento HTML Vazio

    Elementos HTML sem conteúdo são chamados de elementos vazios.

    O elemento <br> define uma quebra de linha e é um elemento vazio sem uma tag de fechamento.

    Exemplo

                        <p>Este é um parágrafo com uma quebra de linha.<br>Esta é uma nova linha no mesmo parágrafo.</p>

    Resultado

    Este é um parágrafo com uma quebra de linha.
    Esta é uma nova linha no mesmo parágrafo.

    Tags HTML Não Diferenciam Maiúsculas de Minúsculas

    As tags HTML não diferenciam maiúsculas de minúsculas: <P> significa o mesmo que <p>.

    Embora o padrão HTML não exija tags em minúsculas, a W3C recomenda o uso de letras minúsculas em HTML e exige isso para documentos mais estritos, como XHTML.
