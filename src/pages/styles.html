• Estilos HTML

    O atributo <style> é usado para adicionar estilos a um elemento, como cor, fonte, tamanho, e mais.

    Exemplo

                        <p style="color: red;">Eu sou vermelho</p>
     <p style="color: blue;">Eu sou azul</p>
     <p style="font-size:30px;">Eu sou grande</p>

    Resultado

    Eu sou vermelho

    Eu sou azul

    Eu sou grande

    Definição do Atributo Style

    Para definir o estilo de um elemento HTML, usamos o atributo style. Ele segue a seguinte sintaxe: 

                        <nome_da_tag style="propriedade:valor;">

    A propriedade é uma propriedade CSS e o valor é um valor CSS.

    Cor de Fundo

    A propriedade CSS background-color define a cor de fundo para um elemento HTML.

    Exemplo

    Define a cor de fundo de uma div como azul claro:

                        <div style="background-color:powderblue;">
        <h1>Este é um título</h1>
        <p>Este é um parágrafo.</p>
     </div>

    Resultado

    Este é um título

    Este é um parágrafo.

    Exemplo

                        <div>
        <h1 style="background-color:powderblue;">Este é um título</h1>
        <p style="background-color:tomato;">Este é um parágrafo.</p>
     </div>

    Resultado

    Este é um título

    Este é um parágrafo.

    Cor do Texto

    A propriedade CSS color define a cor do texto para um elemento HTML:

    Exemplo de Cor do Texto

                        <h1 style="color:blue;">Este é um título</h1>
     <p style="color:red;">Este é um parágrafo.</p>

    Resultado

    Este é um título

    Este é um parágrafo.

    Fontes

    A propriedade CSS font-family define a fonte a ser usada para um elemento HTML:

    Exemplo

                        <h1 style="font-family:Verdana;">Este é um título</h1>
     <p style="font-family:Courier;">Este é um parágrafo.</p>

    Resultado

    Este é um título

    Este é um parágrafo.

    Tamanho do Texto

    A propriedade CSS font-size define o tamanho do texto para um elemento HTML:

    Exemplo

                        <h1 style="font-size:300%;">Este é um título</h1>
     <p style="font-size:160%;">Este é um parágrafo.</p>

    Resultado

    Este é um título

    Este é um parágrafo.

    Alinhamento do Texto

    A propriedade CSS text-align define o alinhamento horizontal do texto para um elemento HTML:

    Exemplo

                        <h1 style="text-align:left;">Título alinhado à esquerda</h1>
     <p style="text-align:left;">Parágrafo alinhado à esquerda.</p>

    Resultado

    Título alinhado à esquerda

    Parágrafo alinhado à esquerda.
