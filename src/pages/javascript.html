• JavaScript em HTML

    O JavaScript torna as páginas HTML mais dinâmicas e interativas.

    Exemplo

                        <button type="button" onclick="document.getElementById('data').innerHTML = Date()">Clique para exibir Data e Hora</button>
     <p id="data"></p>

    Resultado

    A Tag <script> em HTML

    A tag <script> é usada para definir um script no lado do cliente (JavaScript).

    O elemento <script> pode conter instruções de script ou apontar para um arquivo de script externo através do atributo src.

    Usos comuns do JavaScript incluem manipulação de imagens, validação de formulários e alterações dinâmicas de conteúdo.

    Para selecionar um elemento HTML, o JavaScript mais frequentemente usa o método document.getElementById().

    Exemplo de JavaScript Mudando Conteúdo

                        <p id="demo"></p>

     <script>
        document.getElementById("demo").innerHTML = "Olá JavaScript!";
     </script>

    Resultado

    O Que JavaScript Pode Fazer

    Aqui estão alguns exemplos do que o JavaScript pode fazer:

    Exemplo de Mudança de Conteúdo

                        <button type="button" onclick="myFunction1()">Clique Aqui!</button>
     <p id="teste">Isto é uma demonstração.</p>
                    
     <script>
        function myFunction1() { 
        document.getElementById("teste").innerHTML = "Olá JavaScript!";
        }
     </script>

    Resultado

    Isto é uma demonstração.

    Exemplo de Mudança de Estilos

                        <p id="teste2">JavaScript pode mudar o estilo de um elemento HTML.</p>

     <script>
        function myFunction() {
        document.getElementById("teste2").style.fontSize = "25px";
        document.getElementById("teste2").style.color = "red";
        document.getElementById("teste2").style.backgroundColor = "yellow";
        }
     </script>

     <button type="button" onclick="myFunction()">Clique Aqui!</button>

    Resultado

    JavaScript pode mudar o estilo de um elemento HTML.

    Exemplo de Mudança de Atributos

                        <script>
        function light(sw) {
            var pic;
            if (sw == 0) {
            pic = "pic_bulboff.gif";
            } else {
            pic = "pic_bulbon.gif";
            }
            document.getElementById('myImage').src = pic;
        }
     </script>

     <img id="myImage" src="pic_bulboff.gif" width="100" height="180">

     <p>
        <button type="button" onclick="light(1)">Ligar Luz</button>
        <button type="button" onclick="light(0)">Desligar Luz</button>
     </p>

    Resultado

    A Tag <noscript> em HTML

    A tag <noscript> define um conteúdo alternativo para ser exibido aos usuários que desativaram scripts em seus navegadores ou que têm um navegador que não suporta scripts:

    Exemplo de Uso do <noscript>

                        <p id="teste3"></p>

     <script>
        document.getElementById("teste3").innerHTML = "Olá JavaScript!";
     </script>
    
     <noscript>Desculpe, seu navegador não suporta JavaScript!</noscript>

    Resultado

    Tags de Script em HTML

    Tag Descrição
    <script> Define um script no lado do cliente
    <noscript> Define conteúdo alternativo para usuários que não suportam scripts no lado do cliente
