• HTML Vídeo

    O elemento HTML <video> é usado para exibir um vídeo em uma página da web.

    Exemplo

                        <video width="440" height="440" controls>
        <source src="../media/residentEvil.mp4" type="video/mp4">
        Seu navegador não suporta a tag de vídeo.
     </video>

    Resultado

    Como Funciona

    O atributo controls adiciona controles de vídeo, como play, pause e volume.

    É uma boa prática sempre incluir os atributos width e height. Se a largura e a altura não forem definidas, a página pode piscar enquanto o vídeo é carregado.

    O elemento <source> permite especificar arquivos de vídeo alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

    O texto entre as tags <video> e </video> será exibido apenas em navegadores que não suportam o elemento <video>.

    HTML <video> Autoplay

    Para iniciar um vídeo automaticamente, use o atributo autoplay:

    Exemplo

                        <video width="440" height="440" autoplay>
        <source src="../media/residentEvil.mp4" type="video/mp4">
        Seu navegador não suporta a tag de vídeo.
     </video>

    Resultado

    Nota: Navegadores baseados em Chromium geralmente não permitem autoplay na maioria dos casos. No entanto, autoplay com mudo é sempre permitido.

    Adicione o atributo muted após autoplay para permitir que seu vídeo comece a tocar automaticamente (mas sem som):

    Exemplo

                            <video width="440" height="440" autoplay muted>
            <source src="../media/residentEvil.mp4" type="video/mp4">
            Seu navegador não suporta a tag de vídeo.
         </video>

    Resultado

    HTML Vídeo - Métodos, Propriedades e Eventos

    O HTML DOM define métodos, propriedades e eventos para o elemento <video>.

    Isso permite carregar, reproduzir e pausar vídeos, bem como definir duração e volume.

    Existem também eventos DOM que podem notificá-lo quando um vídeo começa a tocar, é pausado, etc.

    Exemplo

                            <button onclick="playPause()">Play/Pause</button>
     <button onclick="makeBig()">Big</button>
     <button onclick="makeSmall()">Small</button>
     <button onclick="makeNormal()">Normal</button>

     <video id="video1" width="440">
        <source src="../media/residentEvil.mp4" type="video/mp4">
        Seu navegador não suporta a tag de vídeo.
     </video>

     <script>
        var myVideo = document.getElementById("video1");

        function playPause() {
            if (myVideo.paused) {
                myVideo.play();
            } else {
                myVideo.pause();
            }
        }

        function makeBig() {
            myVideo.width = 560;
        }

        function makeSmall() {
            myVideo.width = 320;
        }

        function makeNormal() {
            myVideo.width = 420;
        }
    </script>

    Resultado

    Tags de Vídeo HTML

    Tag Descrição
    <video> Define um vídeo ou filme
    <source> Define múltiplos recursos de mídia para elementos de mídia, como <video> e <audio>.
    <track> Define faixas de texto em players de mídia.
