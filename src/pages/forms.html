• Formulários HTML

    Um formulário HTML é usado para coletar dados do usuário. Os dados do usuário são geralmente enviados para um servidor para processamento.

    Exemplo de formulário básico







    O Elemento <form>

    O elemento HTML <form> é usado para criar um formulário para entrada de dados pelo usuário:

    Exemplo

                        <form>
        <p>Coloque aqui os elementos do formulário.</p>
     </form>

    Resultado

    Coloque aqui os elementos do formulário.

    O elemento <form> é um contêiner para diferentes tipos de elementos de entrada, como: campos de texto, checkboxes, botões de rádio, botões de envio, entre outros.

    O Elemento <input>

    O elemento HTML <input> é o elemento mais utilizado em formulários.

    Um elemento <input> pode ser exibido de várias maneiras, dependendo do atributo type.

    Aqui estão alguns exemplos:

    Tipo Descrição
    <input type="text"> Exibe um campo de entrada de linha única para texto
    <input type="radio"> Exibe um botão de rádio (para selecionar uma entre várias opções)
    <input type="checkbox"> Exibe uma checkbox (para selecionar zero ou mais opções entre várias)
    <input type="submit"> Exibe um botão para enviar o formulário
    <input type="button"> Exibe um botão clicável

    Botões de Rádio

    Os botões de rádio permitem que um usuário selecione UMA opção dentre várias.

    Exemplo

                        <form>
        <input type="radio" id="html" name="fav_language" value="HTML"/>
        <label for="html">HTML</label><br>
        <input type="radio" id="css" name="fav_language" value="CSS"/>
        <label for="css">CSS</label><br>
        <input type="radio" id="javascript" name="fav_language" value="JavaScript"/>
        <label for="javascript">JavaScript</label>
     </form>

    Resultado



    Caixas de Seleção (Checkboxes)

    As caixas de seleção permitem que um usuário selecione ZERO ou MAIS opções dentre várias.

    Exemplo

                        <form>
        <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike"/>
        <label for="vehicle1"> Eu tenho uma bicicleta</label><br>
        <input type="checkbox" id="vehicle2" name="vehicle2" value="Car"/>
        <label for="vehicle2"> Eu tenho um carro</label><br>
        <input type="checkbox" id="vehicle3" name="vehicle3" value="Boat"/>
        <label for="vehicle3"> Eu tenho um barco</label>
    </form>

    Resultado



    Botão de Envio

    O botão de envio é usado para enviar dados do formulário para um processador de formulários no servidor.

    Exemplo

                        <form action="/action_page.php">
        <label for="fname">Nome:</label><br>
        <input type="text"id="fname" name="fname" value="John" /><br><br>
        <label for="lname">Sobrenome:</label><br>
        <input type="text" id="lname" name="lname" value="Doe" /><br><br>
        <input type="submit" value="Enviar" />
     </form>

    Resultado
