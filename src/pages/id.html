• Atributo ID em HTML

    O atributo id é usado para especificar um identificador único para um elemento HTML.

    Você não pode ter mais de um elemento com o mesmo id em um documento HTML.

    O valor do atributo id deve ser único dentro do documento HTML.

    O atributo id é usado para apontar para uma declaração de estilo específica em uma folha de estilo. Ele também pode ser usado por JavaScript para acessar e manipular o elemento com o id específico.

    Como Usar o Atributo id

    A sintaxe para usar id é: escreva um caractere de hash (#), seguido por um nome de id. Em seguida, defina as propriedades do CSS entre chaves { }.

    Exemplo de Uso do Atributo id

                        <!DOCTYPE html>
     <html>
     <head>
     <style>
        #myHeader {
            background-color: lightblue;
            color: black;
            padding: 40px;
            text-align: center;
        }
     </style>
     </head>
            
     <body>
        <h2>O Atributo id</h2>
        <p>Use CSS para estilizar um elemento com o id "myHeader":</p>
        <h1 id="myHeader">Meu Cabeçalho</h1>
     </body>
     </html>

    Resultado

    O Atributo id

    Use CSS para estilizar um elemento com o id "myHeader":

    Meu Cabeçalho

    Nota: O nome do id é sensível a maiúsculas e minúsculas!

    Nota: O nome do id deve conter pelo menos um caractere, não pode começar com um número e não deve conter espaços em branco (espaços, tabulações, etc.).

    Diferença Entre Classe e ID

    Um nome de classe pode ser usado por vários elementos HTML, enquanto um nome de id deve ser usado apenas por um elemento HTML na página:

    Exemplo

                        <style>
        /* Estilize o elemento com o id "myHeader" */
        #myHeader {
            background-color: lightblue;
            color: black;
            padding: 40px;
            text-align: center;
        }

        /* Estilize todos os elementos com o nome da classe "city" */
        .city {
            background-color: tomato;
            color: white;
            padding: 10px;
        }
     </style>

     <!-- Um elemento com um id único -->
     <h1 id="myHeader">Minhas Cidades</h1>

     <!-- Vários elementos com a mesma classe -->
     <h2 class="city">Londres</h2>
     <p>Londres é a capital da Inglaterra.</p>

     <h2 class="city">Paris</h2>
     <p>Paris é a capital da França.</p>

     <h2 class="city">Tóquio</h2>
     <p>Tóquio é a capital do Japão.</p>

    Resultado

    Minhas Cidades

    Londres

    Londres é a capital da Inglaterra.

    Paris

    Paris é a capital da França.

    Tóquio

    Tóquio é a capital do Japão.

    Marcadores HTML com ID e Links

    Marcadores HTML são usados para permitir que os leitores pulem para partes específicas de uma página da web.

    Os marcadores podem ser úteis se sua página for muito longa.

    Para usar um marcador, primeiro você deve criá-lo e depois adicionar um link para ele.

    Quando o link é clicado, a página rola para a localização do marcador.

    Exemplo de Marcador HTML

    Parte 1

                        <h2 id="C4">Capítulo 4</h2>

    Resultado

    Capítulo 4

    Parte 2

    Depois, adicione um link para o marcador ("Ir para o Capítulo 4"), a partir da mesma página:

                        <a href="#C4">Ir para o Capítulo 4</a>

    Resultado

    Ir para o Capítulo 4

    Exemplo

    Ou, adicione um link para o marcador ("Ir para o Próximo Tutorial"), a partir de outra página:

                        <a href="iframes.html#HTML Iframes">Ir para o Próximo Tutorial</a>

    Resultado

    Ir para o Próximo Tutorial

    Usando o Atributo id em JavaScript

    O atributo id também pode ser usado por JavaScript para realizar certas tarefas para um elemento específico.

    JavaScript pode acessar um elemento com um id específico usando o método getElementById():

    Exemplo

                        <h1 id="myHeader">Olá Mundo!</h1>
     <button onclick="displayResult()">Mudar texto</button>

     <script>
        function displayResult() {
            document.getElementById("myHeader").innerHTML = "Tenha um bom dia!";
        }
     </script>

    Resultado
