• O Atributo charset no HTML

    Para exibir uma página HTML corretamente, um navegador da Web deve saber qual conjunto de caracteres usar.

    A especificação do HTML5 incentiva os desenvolvedores da web a usar o conjunto de caracteres UTF-8.

    O UTF-8 cobre quase todos os caracteres e símbolos no mundo!

    O conjunto de caracteres é especificado na tag <meta>:

    Exemplo

                        <p><code><meta charset="UTF-8"></code></p>

    Resultado

    Esse site foi feito utilizando o <meta charset="UTF-8">

    O Conjunto de Caracteres ASCII

    O ASCII foi o primeiro padrão de codificação de caracteres para a web. Ele definiu 128 caracteres diferentes que poderiam ser usados na internet:

    Letras em inglês (A-Z)

    Números (0-9)

    Caracteres especiais como ! $ + - ( ) @ < >

    O Conjunto de Caracteres ANSI

    ANSI (Windows-1252) foi o conjunto de caracteres original do Windows:

    Idêntico ao ASCII para os primeiros 127 caracteres

    Caracteres especiais de 128 a 159

    Idêntico ao UTF-8 de 160 a 255

    Exemplo

                        <p><code><meta charset="Windows-1252"></code></p>

    O Conjunto de Caracteres ISO-8859-1

    ISO-8859-1 foi o conjunto de caracteres padrão para HTML 4. Este conjunto suportava 256 códigos diferentes de caracteres. HTML 4 também suportava UTF-8.

    Idêntico ao ASCII para os primeiros 127 caracteres

    Não usa caracteres de 128 a 159

    Idêntico ao ANSI e UTF-8 de 160 a 255

    Exemplo

                        <p><code><meta http-equiv="Content-Type" content="text/html;charset=ISO-8859-1"></code></p>

    O Conjunto de Caracteres UTF-8

    Idêntico ao ASCII para valores de 0 a 127

    Não usa caracteres de 128 a 159

    Idêntico ao ANSI e ISO-8859-1 de 160 a 255

    Continua de 256 até mais de 10.000 caracteres

    Exemplo

                        <p><code><meta charset="UTF-8"></code></p>
