• Elementos de Bloco e Inline em HTML

    Todo elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento.

    Os valores mais comuns são block e inline.

    Elementos de Bloco

    Um elemento de bloco sempre começa em uma nova linha, e os navegadores automaticamente adicionam espaço (margem) antes e depois do elemento.

    Um elemento de bloco sempre ocupa toda a largura disponível (estendendo-se para a esquerda e para a direita tanto quanto possível).

    Dois elementos de bloco comumente usados são: <p> e <div>.

    O elemento define um parágrafo em um Documento HTML. <p>.

    O elemento define uma divisão ou uma seção em um documento HTML. <div>.

    O elemento <p> é um elemento de nível de bloco.

    O elemento <div> é um elemento de nível de bloco.

    Exemplo

                        <p>Hello World</p>
     <div>Hello World</div>

    Resultado

    Hello World

    Hello World

    Elementos Inline

    Um elemento inline não começa em uma nova linha.

    Ele ocupa apenas a largura necessária.

    Este é um elemento <span> dentro de um parágrafo.

    Exemplo de Elementos Inline

                        <span>Hello World</span>

    Resultado

    Hello World

    Nota: Um elemento inline não pode conter um elemento de bloco!

    O Elemento <div>

    O elemento <div> é frequentemente usado como um contêiner para outros elementos HTML.

    Ele não tem atributos obrigatórios, mas style, class e id são comuns.

    Exemplo com o Elemento <div>

                        <div style="background-color:black;color:white;padding:20px;">
        <h2>Londres</h2>
        <p>
            Londres é a capital da Inglaterra. É a cidade mais populosa do Reino Unido, com uma área metropolitana de mais 
            de 13 milhões de habitantes.
        </p>
     </div>

    Resultado

    Londres

    Londres é a capital da Inglaterra. É a cidade mais populosa do Reino Unido, com uma área metropolitana de mais de 13 milhões de habitantes.

    O Elemento <span>

    O elemento <span> é um contêiner inline usado para marcar uma parte do texto ou de um documento.

    Ele não tem atributos obrigatórios, mas style, class, e id são comuns.

    Exemplo com o Elemento <span>

                        <p>
   Minha mãe tem <span style="color:blue;font-weight:bold;">olhos azuis</span> e meu pai tem <span style="color:darkolivegreen;font-weight:bold;">olhos verde-escuros</span> 
     </p>

    Minha mãe tem olhos azuis e meu pai tem olhos verde-escuros

    Tags de Tabelas em HTML

    Tag Descrição
    <div> Define uma seção em um documento (nível de bloco)
    <span> Define uma seção em um documento (inline)
