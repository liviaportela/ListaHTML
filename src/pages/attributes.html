• Atributos HTML

    Os atributos HTML fornecem informações adicionais sobre os elementos HTML.

    Todos os elementos HTML podem ter atributos.

    Os atributos fornecem informações adicionais sobre os elementos.

    Os atributos são sempre especificados na tag de abertura.

    Os atributos geralmente vêm em pares nome/valor, como: nome="valor".

    O Atributo href

    A tag <a> define um hiperlink. O atributo href especifica o URL para o qual o link aponta.

    Exemplo

                        <a href="https://example.com" target="_blank">Link para Example.com</a>

    Resultado

    Link para Example.com

    O Atributo src

    A tag <img> é usada para incorporar uma imagem em uma página HTML. O atributo src especifica o caminho para a imagem a ser exibida.

    Exemplo

                        <img src="../media/Dog.png">

    Resultado

    Há duas maneiras de especificar o URL no atributo src:

    URL absoluto - Aponta para uma imagem externa hospedada em outro site. Exemplo: src="https://i.pinimg.com/originals/8d/5b/ea/8d5bea80a05662e69d6acc9ef10bace8.jpg".
    URL relativo - Aponta para uma imagem hospedada dentro do site. Exemplo: src="../media/Dog.png".

    Os Atributos width e height

    A tag <img> também deve conter os atributos width e height, que especificam a largura e altura da imagem (em pixels):

    Exemplo

                        <img src="../media/Dog.png" width="300" height="300">

    Resultado

    O Atributo alt

    O atributo alt é obrigatório para a tag <img> e especifica um texto alternativo para a imagem, caso a imagem não possa ser exibida:

    Exemplo

                        <img src="../media/Dog.png" alt="Cachorro" width="300" height="300">

    Resultado

    O Atributo style

    O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho, etc.

    Exemplo

                        <p style="color: red;">Este é um parágrafo vermelho.</p>

    Resultado

    Este é um parágrafo vermelho.

    O Atributo lang

    O atributo lang deve ser incluído na tag <html> para declarar o idioma da página da web: 

                    <!DOCTYPE html>
        <html lang="pt-BR">
        <body>
            ...
        </body>
        </html>

    O Atributo title

    O atributo title define uma informação adicional sobre um elemento. O valor do atributo será exibido como uma dica (tooltip) quando você passar o mouse sobre o elemento:

    Exemplo

                        <p title="Eu sou uma dica">Este é um parágrafo.</p>

    Resultado

    Este é um parágrafo.

    Recomendações sobre Atributos HTML

    Sempre use atributos em letras minúsculas. Embora o padrão HTML não exija atributos em minúsculas, a W3C recomenda isso.

    Sempre coloque aspas nos valores dos atributos. Mesmo que o padrão HTML não exija, é uma boa prática.

    Aspas simples ou duplas?

    Você pode usar aspas simples ou duplas para envolver valores de atributos, mas aspas duplas são mais comuns.

    Em alguns casos, onde o valor do atributo contém aspas duplas, você deve usar aspas simples.

    Exemplo

                        <p title="John 'ShotGun' Nelson">Este é um parágrafo.</p>

    Ou vice-versa:

                        <p title='John "ShotGun" Nelson'>Este é um parágrafo.</p>

    Resultado

    Este é um parágrafo.

    Este é um parágrafo.
