• HTML Favicon

    O favicon é um ícone que aparece na aba do navegador para identificar a página.

    Exemplo

                        <head>
        <title>Título da Minha Página</title>
        <link rel="shortcut icon" href="../media/favicon.png" type="image/x-icon">
     </head>

    Resultado

    O favicon é visível na aba do navegador.

    Você pode usar qualquer imagem como seu favicon. Também é possível criar seu próprio favicon em sites como favicon.cc.

    Dica: Um favicon é uma imagem pequena, portanto deve ser uma imagem simples com alto contraste.

    Para adicionar um favicon ao seu site, salve a imagem do favicon no diretório raiz do seu servidor web, ou crie uma pasta chamada "imagens" no diretório raiz e salve a imagem do favicon nessa pasta. Um nome comum para a imagem do favicon é "favicon.ico".
