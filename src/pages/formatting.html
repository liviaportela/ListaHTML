• Elementos de Formatação HTML

    O HTML contém vários elementos para definir texto com um significado especial.

    Exemplo

                        <b>Este texto é em negrito</b>
     <i>Este texto é em itálico</i>
     <p>Este é subescrito e sobrescrito: H<sub>2</sub>O é água, e 1<sup>º</sup> é primeiro.</p>

    Resultado

    Este texto é em negrito Este texto é em itálico

    Este é subescrito e sobrescrito: H2O é água, e 1º é primeiro.

    Elementos de Formatação HTML

    Os elementos de formatação foram projetados para exibir tipos especiais de texto:

    • <b> - Texto em negrito
    • <strong> - Texto importante
    • <i> - Texto em itálico
    • <em> - Texto enfatizado
    • <mark> - Texto marcado/destacado
    • <small> - Texto menor
    • <del> - Texto deletado
    • <ins> - Texto inserido
    • <sub> - Texto subescrito
    • <sup> - Texto sobrescrito

    Elementos <b> e <strong>

    O elemento HTML <b> define texto em negrito, sem qualquer importância adicional.

    Exemplo

                        <p><b>Este texto é em negrito</b></p>

    Resultado

    Este texto é em negrito

    O elemento HTML <strong> define texto com forte importância. O conteúdo é geralmente exibido em negrito.

    Exemplo

                        <p><strong>Este texto é importante!</strong></p>

    Resultado

    Este texto é importante!

    Elementos <i> e <em>

    O elemento HTML <i> define uma parte do texto em uma voz ou tom alternativo, geralmente exibido em itálico.

    Dica: A tag <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, ou nome de um navio.

    Exemplo

    <p><i>Este texto é em itálico</i></p>

    Resultado

    Este texto é em itálico

    O elemento HTML <em> define texto enfatizado, geralmente exibido em itálico.

    Dica: Um leitor de tela pronunciará as palavras com ênfase ao encontrar o elemento <em>.

    Exemplo

    <p><em>Este texto é enfatizado</em></p>

    Resultado

    Este texto é enfatizado

    Elemento <small>

    O elemento HTML <small> define texto menor:

    Exemplo

                        <p>Este é um texto <small>menor.</small></p>

    Resultado

    Este é um texto menor.

    Elemento <mark>

    O elemento HTML <mark> define texto que deve ser marcado ou destacado:

    Exemplo

                        <p>Não se esqueça de comprar <mark>leite</mark> hoje.</p>

    Resultado

    Não se esqueça de comprar leite hoje.

    Elemento <del>

    O elemento HTML <del> define texto que foi deletado de um documento. Os navegadores geralmente exibem texto deletado com uma linha através dele.

    Exemplo

                        <p>Minha cor favorita é <del>azul</del> vermelho.</p>

    Resultado

    Minha cor favorita é azul vermelho.

    Elemento <ins>

    O elemento HTML <ins> define texto que foi inserido em um documento. Os navegadores geralmente sublinham texto inserido.

    Exemplo

                        <p>Minha cor favorita é <del>azul</del> <ins>vermelho</ins>.</p>

    Resultado

    Minha cor favorita é azul vermelho.

    Elemento <sub>

    O elemento HTML <sub> define texto subescrito, que aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. Ele pode ser usado para fórmulas químicas, como H2O.

    Exemplo

                        <p>Este é um texto <sub>subescrito</sub>.</p>

    Resultado

    Este é um texto subescrito.

    Elemento <sup>

    O elemento HTML <sup> define texto sobrescrito, que aparece meio caractere acima da linha normal e também pode ser menor. Ele pode ser usado para notas de rodapé, como WWW[1].

    Exemplo

                        <p>Este é um texto <sup>sobrescrito</sup>.</p>

    Resultado

    Este é um texto sobrescrito.
