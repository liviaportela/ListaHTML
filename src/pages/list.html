• Listas em HTML

    Listas em HTML permitem que desenvolvedores da web agrupem um conjunto de itens relacionados.

    Lista Não Ordenada em HTML

    Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>. Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

    Exemplo

                        <ul>
        <li>Café</li>
        <li>Chá</li>
        <li>Leite</li>
     <ul>

    Resultado

    • Café
    • Chá
    • Leite

    Lista Ordenada em HTML

    Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>. Os itens da lista serão marcados com números por padrão:

    Exemplo

                        <ol>
        <li>Café</li>
        <li>Chá</li>
        <li>Leite</li>
     <ol>

    Resultado

    1. Café
    2. Chá
    3. Leite

    Listas de Descrição em HTML

    HTML também suporta listas de descrição. Uma lista de descrição é uma lista de termos com uma descrição para cada termo. A tag <dl> define a lista de descrição, a tag <dt> define o termo (nome), e a tag <dd> descreve cada termo:

    Exemplo

                        <dl>
        <dt>Café</dt>
        <dd>- bebida quente preta</dd>
        <dt>Leite</dt>
        <dd>- bebida fria branca</dd>
     <dl>

    Resultado

    Café
    - bebida quente preta
    Leite
    - bebida fria branca

    Tags para Listas em HTML

    Tag Descrição
    <ul> Define uma lista não ordenada
    <ol> Define uma lista ordenada
    <li> Define um item da lista
    <dl> Define uma lista de descrição
    <dt> Define um termo em uma lista de descrição
    <dd> Descreve o termo em uma lista de descrição
