    Os links são criados com o elemento <a> e podem ser usados para navegação.

                        <a href="https://example.com">Link para Example.com</a>
    Link para Example.com

    Links em HTML - Hiperlinks

    Os links são encontrados em quase todas as páginas da web. Eles permitem aos usuários navegar de uma página para outra.

    Definição de Links

    Links em HTML são hiperlinks.

    Você pode clicar em um link para saltar para outro documento.

    Quando você move o mouse sobre um link, a seta do mouse se transforma em uma mãozinha.

    Sintaxe de Links

    A tag HTML <a> define um hyperlink. A sintaxe é a seguinte:

    Exemplo

                        <a href="url" target="_blank">texto do link</a>

    Resultado

    texto do link

    O atributo mais importante do elemento <a> é o atributo href, que indica o destino do link.

    O texto do link é a parte que será visível para o leitor.

    Clicar no texto do link enviará o leitor para o endereço URL especificado.

    Exemplo

                            <a href="https://www.w3schools.com/">Visite o W3Schools.com!</a>

    Resultado

    Visite o W3Schools.com!

    Por padrão, os links aparecerão da seguinte maneira em todos os navegadores:

    • Um link não visitado é sublinhado e azul.
    • Um link visitado é sublinhado e roxo.
    • Um link ativo é sublinhado e vermelho.

    Dica: Links podem ser estilizados com CSS para ter uma aparência diferente!

    O Atributo target

    Por padrão, a página vinculada será exibida na janela/aba atual do navegador. Para mudar isso, você deve especificar um destino diferente para o link. O atributo target pode ter um dos seguintes valores:

    _self - Padrão. Abre o documento na mesma janela/aba em que foi clicado.

    _blank - Abre o documento em uma nova janela ou aba.

    _parent - Abre o documento no quadro pai.

    _top - Abre o documento no corpo completo da janela.

    Exemplo

                        <a href="https://www.w3schools.com/" target="_blank">Visite o W3Schools.com!</a>

    Resultado

    Visite o W3Schools.com!

    URLs Absolutas vs. URLs Relativas

    Ambos os exemplos anteriores usam um URL absoluto (um endereço web completo) no atributo href.

    Um link local (um link para uma página dentro do mesmo site) é especificado com um URL relativo (sem a parte "https://www"):

    Exemplo de URLs Absolutas

                            <a href="https://www.w3.org/" target="_blank">W3C</a>
     <a href="https://www.google.com/" target="_blank">Google</a>

    Resultado

    W3C

    Google

    Exemplo de URLs Relativas

                            <a href="/ListaHTML/src/media/" target="_blank">Imagens HTML</a>
     <a href="/ListaHTML/src/pages/favicon.html" target="_blank">Próximo tutorial</a>

    Resultado

    Imagens HTML

    Próximo tutorial

    Usar uma Imagem como Link

    Para usar uma imagem como link, basta colocar a tag <img> dentro da tag <a>:

    Exemplo

                        <a href="https://www.w3schools.com/"> 
        <img src="../media/Dog.png" alt="Próximo Tutorial" style="width:42px;height:42px;">
     </a>

    Resultado

    Link para um Endereço de E-mail

    Use mailto: dentro do atributo href para criar um link que abre o programa de e-mail do usuário para enviar um novo e-mail:

    Exemplo

                        <a href="mailto:someone@example.com">Enviar e-mail</a>

    Resultado

    Enviar e-mail

    Botão como Link

    Para usar um botão HTML como link, você precisa adicionar algum código JavaScript. JavaScript permite especificar o que acontece em determinados eventos, como um clique em um botão:

    Exemplo

                        <button onclick="document.location='/ListaHTML/src/pages/favicon.html'">Tutorial de HTML</button>

    Resultado

    O Atributo title em Links

    O atributo title especifica informações extras sobre um elemento. Essas informações geralmente são exibidas como uma dica de ferramenta (tooltip) quando o mouse passa sobre o elemento.

    Exemplo

                        <a href="https://www.w3schools.com/html/" title="Ir para a seção HTML do W3Schools">Visite nosso Tutorial de HTML</a>

    Resultado

    Visite nosso Tutorial de HTML

    Mais Sobre URLs Absolutas e Relativas

    Exemplo de URL Absoluta

    Use um URL completo para vincular a uma página da web:

                        <a href="https://www.w3schools.com/html/default.asp" target="_blank">Tutorial de HTML</a>

    Resultado

    Tutorial de HTML

    Exemplo de URL Relativa para a Mesma Pasta

    Link para uma página localizada na mesma pasta que a página atual:

                        <a href="favicon.html" target="_blank">Visite o W3Schools.com!</a>

    Resultado

    Tutorial de HTML
