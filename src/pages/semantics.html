• Elementos Semânticos em HTML

    Os elementos semânticos são elementos que têm significado.

    Um elemento semântico descreve claramente seu significado para o navegador e para o desenvolvedor.

    Exemplos de elementos não semânticos: <div> e <span> - Não dizem nada sobre seu conteúdo.

    Exemplos de elementos semânticos: <form>, <table>, e <article> - Definem claramente seu conteúdo.

    Elementos Semânticos em HTML

    Muitos sites contêm código HTML como: <div id="nav">, <div class="header">, <div id="footer"> para indicar navegação, cabeçalho e rodapé.

    No HTML, existem alguns elementos semânticos que podem ser usados para definir diferentes partes de uma página da web:

    • <article>
    • <aside>
    • <details>
    • <figcaption>
    • <figure>
    • <footer>
    • <header>
    • <main>
    • <mark>
    • <nav>
    • <section>
    • <summary>
    • <time>

    Exemplo do Elemento <section>

    O elemento <section> define uma seção em um documento. Segundo a documentação HTML do W3C: "Uma seção é um agrupamento temático de conteúdo, normalmente com um cabeçalho."

                        <section>
        <h1>WWF</h1>
        <p>
            O Fundo Mundial para a Natureza (WWF) é uma organização internacional que trabalha em questões relacionadas à conservação, pesquisa e restauração do 
            meio ambiente, anteriormente chamada de Fundo Mundial para a Vida Selvagem. O WWF foi fundado em 1961.
        </p>
     </section>
        
     <section>
        <h1>O símbolo do Panda do WWF</h1>
        <p>
            O Panda se tornou o símbolo do WWF. O famoso logo do panda do WWF teve origem em um panda chamado Chi Chi, que foi transferido do Zoológico de Pequim 
            para o Zoológico de Londres no mesmo ano da criação do WWF.
        </p>
     </section>

    Resultado

    WWF

    O Fundo Mundial para a Natureza (WWF) é uma organização internacional que trabalha em questões relacionadas à conservação, pesquisa e restauração do meio ambiente, anteriormente chamada de Fundo Mundial para a Vida Selvagem. O WWF foi fundado em 1961.

    O símbolo do Panda do WWF

    O Panda se tornou o símbolo do WWF. O famoso logo do panda do WWF teve origem em um panda chamado Chi Chi, que foi transferido do Zoológico de Pequim para o Zoológico de Londres no mesmo ano da criação do WWF.

    Exemplo do Elemento <article>

    O elemento <article> especifica conteúdo independente e autossuficiente.

    Um artigo deve fazer sentido por conta própria e deve ser possível distribuí-lo independentemente do resto do site.

                        <article>
        <h2>Google Chrome</h2>
        <p>
            Google Chrome é um navegador web desenvolvido pela Google, lançado em 2008. Chrome é o navegador web mais popular do mundo hoje!
        </p>
     </article>
    
     <article>
        <h2>Mozilla Firefox</h2>
        <p>
            Mozilla Firefox é um navegador web de código aberto desenvolvido pela Mozilla. Firefox tem sido o segundo navegador web mais popular desde janeiro de 
            2018.
        </p>
     </article>
    
     <article>
        <h2>Microsoft Edge</h2>
        <p>
            Microsoft Edge é um navegador web desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.
        </p>
     </article>

    Resultado

    Google Chrome

    Mozilla Firefox

    Mozilla Firefox é um navegador web de código aberto desenvolvido pela Mozilla. Firefox tem sido o segundo navegador web mais popular desde janeiro de 2018.

    Microsoft Edge

    Microsoft Edge é um navegador web desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.

    Exemplo do Elemento <header>

    O elemento <header> representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

                        <article>
        <header>
            <h1>O que o WWF faz?</h1>
            <p>A missão do WWF:</p>
        </header>
        <p>
            A missão do WWF é parar a degradação do ambiente natural do nosso planeta e construir um futuro em que os seres humanos vivam em harmonia com a natureza.
        </p>
     </article>

    Resultado

    O que o WWF faz?

    A missão do WWF:

    A missão do WWF é parar a degradação do ambiente natural do nosso planeta e construir um futuro em que os seres humanos vivam em harmonia com a natureza.

    Exemplo do Elemento <footer>

    O elemento <footer> define um rodapé para um documento ou seção.

                        <footer>
        <p>Autor: Hege Refsnes</p>
        <p><a href="mailto:hege@example.com">hege@example.com</a></p>
     </footer>

    Resultado

    Autor: Hege Refsnes

    hege@example.com

    Exemplo do Elemento <nav>

    O elemento <nav> define um conjunto de links de navegação.

    É importante notar que nem todos os links de um documento devem estar dentro de um <nav>. O elemento <nav> é destinado apenas para grandes blocos de links de navegação. 

                        <nav>
        <a href="/html/">HTML</a> |
        <a href="/css/">CSS</a> |
        <a href="/js/">JavaScript</a> |
        <a href="/jquery/">jQuery</a>
     </nav>

    Resultado

    HTML | CSS | JavaScript | jQuery

    Exemplo do Elemento <aside>

    O elemento <aside> define algum conteúdo separado do conteúdo em que está inserido (como uma barra lateral).

    O conteúdo de <aside> deve ser indiretamente relacionado ao conteúdo ao seu redor. 

                        <style>
        aside {
            width: 30%;
            padding-left: 15px;
            margin-left: 15px;
            float: right;
            font-style: italic;
            background-color: lightgray;
        }
     </style>

     <aside>
        <p>
            O centro Epcot é um parque temático no Walt Disney World Resort, com atrações emocionantes, pavilhões internacionais, fogos de artifício premiados
             e eventos especiais sazonais.
        </p>
     </aside>

    Exemplo do Elemento <figure> com <figcaption>

    O elemento <figure> especifica conteúdo autossuficiente, como ilustrações, diagramas, fotos, listas de código, etc.

    O elemento <figcaption> define uma legenda para um <figure>. A legenda pode ser colocada como o primeiro ou o último filho do elemento <figure>. 

                        <figure>
        <img src="../media/Dog.png" alt="Cachorro">
        <figcaption>Fig.1 - Cachorro.</figcaption>
     </figure>

    Resultado

    Por Que Elementos Semânticos?

    De acordo com o W3C: "Uma Web semântica permite que os dados sejam compartilhados e reutilizados em diferentes aplicativos, empresas e comunidades."

    Lista de Elementos Semânticos em HTML

    Tag Descrição
    <article> Define conteúdo independente e autossuficiente
    <aside> Define conteúdo separado do conteúdo principal
    <details> Define detalhes adicionais que o usuário pode ver ou ocultar
    <figcaption> Define uma legenda para um elemento <figure>
    <figure> Especifica conteúdo autossuficiente, como ilustrações, diagramas, fotos, listas de código, etc.
    <footer> Define um rodapé para um documento ou seção
    <header> Especifica um cabeçalho para um documento ou seção
    <main> Especifica o conteúdo principal de um documento
    <mark> Define texto destacado
    <nav> Define links de navegação
    <section> Define uma seção em um documento
    <summary> Define um cabeçalho visível para um elemento <details>
    <time> Define uma data/hora
