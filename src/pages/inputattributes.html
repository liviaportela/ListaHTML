• Atributos do Elemento <input>

    Os atributos do input controlam o comportamento do campo de entrada.

    O Atributo value

    O atributo value define um valor inicial para um campo de entrada.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label>
        <input type="text" id="fname" name="fname" value="John"/>
        <br>
        <label for="lname">Último nome:</label>
        <input type="text" id="lname" name="lname" value="Doe"/>
        <br>
        <input type="submit" value="Enviar"/>
     </form>

    Resultado




    O Atributo readonly

    O atributo readonly define que um campo de entrada é somente leitura.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label>
        <input type="text" id="fname" name="fname" value="John" readonly/>
        <label for="lname">Último nome:</label>
        <input type="text" id="lname" name="lname" value="Doe"/>
     </form>

    Resultado




    O Atributo disabled

    O atributo disabled define que um campo de entrada está desativado e não pode ser usado.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" value="John" disabled/><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname" value="Doe"/>
     </form>

    Resultado




    O Atributo size

    O atributo size especifica a largura visível, em caracteres, de um campo de entrada.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" size="50"/><br>
        <label for="pin">PIN:</label><br>
        <input type="text" id="pin" name="pin" size="4"/>
     </form>

    Resultado




    O Atributo maxlength

    O atributo maxlength define o número máximo de caracteres permitidos em um campo de entrada.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" maxlength="50" size="50"/><br>
        <label for="pin">PIN:</label><br>
        <input type="text" id="pin" name="pin" maxlength="4" size="4"/>
     </form>

    Resultado




    Os Atributos min e max

    Os atributos min e max definem os valores mínimos e máximos para um campo de entrada.

    Exemplo

                        <form>
        <label for="datemax">Insira uma data antes de 1980-01-01:</label>
        <input type="date" id="datemax" name="datemax" max="1979-12-31"/> <br><br>
        <label for="datemin">Insira uma data após 2000-01-01:</label>
        <input type="date" id="datemin" name="datemin" min="2000-01-02"/> <br><br>
        <label for="quantity">Quantidade (entre 1 e 5):</label>
        <input type="number" id="quantity" name="quantity" min="1" max="5"/>
     </form>

    Resultado





    O Atributo multiple

    O atributo multiple permite ao usuário inserir mais de um valor em um campo de entrada.

    Exemplo

                        <form>
        <label for="files">Selecione arquivos:</label>
        <input type="file" id="files" name="files" multiple/>
     </form>

    Resultado

    O Atributo pattern

    O atributo pattern especifica uma expressão regular contra a qual o valor do campo de entrada é verificado ao enviar o formulário.

    Exemplo

                        <form>
        <label for="country_code">Código do país:</label>
        <input type="text" id="country_code" name="country_code" pattern="[A-Za-z]{3}" title="Código de país de três letras"/>
     </form>

    Resultado

    O Atributo placeholder

    O atributo placeholder especifica uma dica curta que descreve o valor esperado de um campo de entrada.

    Exemplo

                        <form>
        <label for="phone">Insira um número de telefone:</label>
        <input type="tel" id="phone" name="phone" placeholder="123-45-678" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}"/>
     </form>

    Resultado

    O Atributo required

    O atributo required especifica que um campo de entrada deve ser preenchido antes de enviar o formulário.

    Exemplo

                        <form>
        <label for="username">Nome de usuário:</label>
        <input type="text" id="username" name="username" required/>
     </form>

    Resultado

    O Atributo step

    O atributo step especifica os intervalos de número legais para um campo de entrada.

    Exemplo

                        <form>
        <label for="points">Pontos:</label>
        <input type="number" id="points" name="points" step="3"/>
     </form>

    Resultado

    O Atributo autofocus

    O atributo autofocus especifica que um campo de entrada deve automaticamente receber foco quando a página é carregada.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" autofocus/><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname"/>
     </form>

    Resultado




    Os Atributos height e width

    Os atributos height e width especificam a altura e a largura de um elemento.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label>
        <input type="text" id="fname" name="fname"/> <br><br>
        <label for="lname">Último nome:</label>
        <input type="text" id="lname" name="lname"/> <br><br>
        <input type="image" src="../media/Dog.png" alt="Enviar" width="48" height="48"/>
     </form>

    Resultado





    O Atributo list

    O atributo list refere-se a um elemento <datalist> que contém opções predefinidas para um elemento <input>.

    Exemplo

                        <form>
        <input list="browsers"/>
        <datalist id="browsers">
            <option value="Edge"></option>
            <option value="Firefox"></option>
            <option value="Chrome"></option>
            <option value="Opera"></option>
            <option value="Safari"></option>
        </datalist>
     </form>

    Resultado

    O Atributo autocomplete

    O atributo autocomplete especifica se um formulário ou um campo de entrada deve ter preenchimento automático ativado ou desativado.

    Exemplo

                        <form action="/action_page.php" autocomplete="on">
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname"/><br><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname"/><br><br>
        <label for="email">E-mail:</label><br>
        <input type="email" id="email" name="email" autocomplete="off"/><br><br>
        <input type="submit" value="Enviar"/>
     </form>

    Resultado
