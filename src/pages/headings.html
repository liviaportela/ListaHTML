• Cabeçalhos HTML

    Os cabeçalhos HTML são títulos ou subtítulos que você deseja exibir em uma página da web.

    Eles são definidos com as tags <h1> a <h6>.

    <h1> define o cabeçalho mais importante, enquanto <h6> define o menos importante.

    Exemplo

                        <h1>Cabeçalho 1</h1>
     <h2>Cabeçalho 2</h2>
     <h3>Cabeçalho 3</h3>
     <h4>Cabeçalho 4</h4>
     <h5>Cabeçalho 5</h5>
     <h6>Cabeçalho 6</h6>

    Resultado

    Cabeçalho 1

    Cabeçalho 2

    Cabeçalho 3

    Cabeçalho 4

    Cabeçalho 5
    Cabeçalho 6

    Importância dos Cabeçalhos

    Os motores de busca usam os cabeçalhos para indexar a estrutura e o conteúdo das suas páginas da web.

    Os usuários muitas vezes percorrem uma página pelos cabeçalhos. Portanto, é importante usar cabeçalhos para mostrar a estrutura do documento.

    Os cabeçalhos <h1> devem ser usados para os títulos principais, seguidos pelos cabeçalhos <h2>, e assim por diante, com <h3> sendo menos importantes.

    Nota: Use cabeçalhos HTML apenas para títulos. Não use cabeçalhos para deixar o texto GRANDE ou em negrito.

    Alterando o Tamanho dos Cabeçalhos

    Cada cabeçalho HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho para qualquer cabeçalho com o atributo style, usando a propriedade CSS font-size.

    Exemplo

                        <h1 style="font-size:60px;">Cabeçalho 1</h1>

    Resultado

    Cabeçalho 1
