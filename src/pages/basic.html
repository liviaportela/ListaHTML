• HTML Básico

    Todos os documentos HTML devem começar com uma declaração de tipo de documento: <!DOCTYPE html>.

    O próprio documento HTML começa com <html> e termina com </html>.

    A parte visível do documento HTML está entre <body> e </body>.

    Exemplo

                    <!DOCTYPE html>
        <html lang="pt-BR">
        <body>
            <h1>Meu primeiro título</h1>
            <p>Meu primeiro parágrafo.</p>
        </body>
        </html>

    Resultado

    Meu primeiro título

    Meu primeiro parágrafo.

    A Declaração <!DOCTYPE>

    A declaração <!DOCTYPE> representa o tipo de documento e ajuda os navegadores a exibirem corretamente as páginas da web.

    Deve aparecer apenas uma vez, no topo da página (antes de qualquer tag HTML).

    A declaração <!DOCTYPE> não diferencia maiúsculas de minúsculas.

    A declaração <!DOCTYPE> para HTML5 é:

                        <!DOCTYPE html>

    Títulos HTML

    Os títulos HTML são definidos com as tags <h1> a <h6>.

    <h1> define o título mais importante. <h6> define o título menos importante.

    Exemplo

                        <h1>Este é o título 1</h1>
      <h2>Este é o título 2</h2>
      <h3>Este é o título 3</h3>

    Resultado

    Este é o título 1

    Este é o título 2

    Este é o título 3

    Parágrafos HTML

    Parágrafos HTML são definidos com a tag <p>.

    Exemplo

                        <p>Este é um parágrafo.</p>
      <p>Este é outro parágrafo.</p>

    Resultado

    Este é um parágrafo.

    Este é outro parágrafo.

    Links HTML

    Links HTML são definidos com a tag <a>.

                        <a href="https://example.com" target="_blank">Link para Example.com</a>

    Resultado

    Link para Example.com

    O destino do link é especificado no atributo href.

    Imagens HTML

    Imagens HTML são definidas com a tag <img>.

    O arquivo de origem (src), texto alternativo (alt), largura (width) e altura (height) são fornecidos como atributos: 

                        <img src="../media/Dog.png" alt="Cachorro" width="300" height="300">

    Como Visualizar o Código Fonte HTML

    Já viu uma página da Web e se perguntou "Como eles fizeram isso?"

    Visualizar o Código Fonte HTML:

    Clique em CTRL + U em uma página HTML, ou clique com o botão direito na página e selecione "Exibir Código-fonte da Página". Isso abrirá uma nova guia contendo o código-fonte HTML da página.

    Inspecionar um Elemento HTML:

    Clique com o botão direito em um elemento (ou em uma área em branco) e escolha "Inspecionar" para ver de quais elementos ele é composto (você verá tanto o HTML quanto o CSS). Você também pode editar o HTML ou CSS rapidamente no painel Elementos ou Estilos que se abre.
