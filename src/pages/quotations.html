• Elementos de Citação HTML

    Uma citação serve para atribuir autoridade, referenciar fontes, evitar plágio, ilustrar ou contextualizar ideias em um texto ou discurso.

    Elemento HTML <blockquote> para Citações

    O elemento HTML <blockquote> define uma seção que é citada de outra fonte.

    Os navegadores geralmente aplicam um recuo ao elemento <blockquote>, destacando-o visualmente.

    Exemplo

                        <p>Aqui está uma citação do site da WWF:</p>
     <blockquote cite="http://www.worldwildlife.org/who/index.html"> 
        Por 60 anos, a WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como a maior
        organização de conservação do mundo, a WWF trabalha em quase 100 países. Em todos os níveis,
        colaboramos com pessoas ao redor do mundo para desenvolver e fornecer soluções inovadoras que
        protegem comunidades, vida selvagem e os lugares em que vivem.
     </blockquote>

    Resultado

    Aqui está uma citação do site da WWF:

    Por 60 anos, a WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como a maior organização de conservação do mundo, a WWF trabalha em quase 100 países. Em todos os níveis, colaboramos com pessoas ao redor do mundo para desenvolver e fornecer soluções inovadoras que protegem comunidades, vida selvagem e os lugares em que vivem.

    Elemento HTML <q> para Citações Curtas

    O elemento HTML <q> define uma citação curta.

    Os navegadores normalmente inserem aspas ao redor da citação.

    Exemplo

                        <p>O objetivo da WWF é: <q>Construir um futuro onde as pessoas vivam em harmonia com a natureza.</q></p>

    Resultado

    O objetivo da WWF é: Construir um futuro onde as pessoas vivam em harmonia com a natureza.

    Elemento HTML <abbr> para Abreviações

    O elemento HTML <abbr> define uma abreviação ou acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

    Marcar abreviações pode fornecer informações úteis para navegadores, sistemas de tradução e mecanismos de busca.

    Dica: Use o atributo global title para mostrar a descrição da abreviação/acrônimo ao passar o mouse sobre o elemento.

    Exemplo

                        <p>A <abbr title="Organização Mundial da Saúde">OMS</abbr> foi fundada em 1948.</p>

    Resultado

    A OMS foi fundada em 1948.

    Elemento HTML <address> para Informações de Contato

    O elemento HTML <address> define as informações de contato para o autor/proprietário de um documento ou artigo.

    As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, perfil de mídia social, etc.

    O texto no elemento <address> geralmente é exibido em itálico, e os navegadores sempre adicionarão uma quebra de linha antes e depois do elemento <address>.

    Exemplo

                        <address> 
        Escrito por John Doe.<br>
        Visite-nos em:<br>
        Example.com<br>
        Caixa Postal 564, Disneyland<br>
        EUA
     </address>

    Resultado

    Escrito por John Doe.
    Visite-nos em:
    Example.com
    Caixa Postal 564, Disneyland
    EUA

    Elemento HTML <cite> para Título de Obras

    O elemento HTML <cite> define o título de uma obra criativa (por exemplo, um livro, poema, música, filme, pintura, escultura, etc.).

    Nota: O nome de uma pessoa não é o título de uma obra.

    Exemplo

                        <p><cite>O Grito</cite> por Edvard Munch. Pintado em 1893.</p>

    Resultado

    O Grito por Edvard Munch. Pintado em 1893.

    Elemento HTML <bdo> para Substituição Bidirecional

    BDO significa "Substituição Bidirecional".

    O elemento HTML <bdo> é usado para substituir a direção do texto atual:

    Exemplo

                        <bdo dir="rtl">Este texto será escrito da direita para a esquerda.</bdo>

    Resultado

    Este texto será escrito da direita para a esquerda.

    Referência para Elementos de Citação e Formatação HTML

    Tag Descrição
    <abbr> Define uma abreviação ou acrônimo
    <address> Define informações de contato para o autor/proprietário de um documento
    <bdo> Define a direção do texto
    <blockquote> Define uma seção citada de outra fonte
    <cite> Define o título de uma obra
    <q> Define uma citação curta
