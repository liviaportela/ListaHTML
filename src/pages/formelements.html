• Elementos de <form> em HTML

    Os elementos de formulário incluem campos de texto, botões de opção, caixas de seleção, entre outros.

    O elemento HTML <form> pode conter um ou mais dos seguintes elementos de formulário:

    • <input>
    • <label>
    • <select>
    • <textarea>
    • <button>
    • <fieldset>
    • <legend>
    • <datalist>
    • <output>
    • <option>
    • <optgroup>

    O Elemento <input>

    Um dos elementos de formulário mais utilizados é o elemento <input>.

    O elemento <input> pode ser exibido de várias maneiras, dependendo do atributo type.

    Exemplo

                        <label for="fname">Primeiro nome:</label>
     <input type="text" id="fname" name="fname"/>

    Resultado

    O Elemento <label>

    O elemento <label> define um rótulo para vários elementos de formulário.

    O elemento <label> é útil para usuários com leitores de tela, porque o leitor de tela lerá o rótulo em voz alta quando o usuário focar no elemento de entrada.

    Além disso, o elemento <label> ajuda usuários com dificuldade para clicar em áreas pequenas (como botões de rádio ou caixas de seleção), pois ao clicar no texto dentro do <label>, ele alterna o botão de rádio/caixa de seleção.

    O atributo for do <label> deve ser igual ao atributo id do elemento <input> para vinculá-los.

    O Elemento <select>

    O elemento <select> define uma lista suspensa:

    Exemplo

                        <label for="cars">Escolha um carro:</label>
     <select id="cars" name="cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
        <option value="fiat">Fiat</option>
        <option value="audi">Audi</option>
     </select>

    Resultado

    O Elemento <textarea>

    O elemento <textarea> define um campo de entrada de várias linhas (uma área de texto):

    Exemplo

                        <textarea name="message" rows="10" cols="30" >O gato estava brincando no jardim.</textarea>

    Resultado

    O Elemento <button>

    O elemento <button> define um botão clicável:

    Exemplo

                        <button type="button" onclick="alert('Olá Mundo!')">Clique em mim!</button>

    Resultado

    O Elemento <fieldset> e <legend>

    O elemento <fieldset> é usado para agrupar dados relacionados em um formulário.

    O elemento <legend> define uma legenda para o elemento <fieldset>.

    Exemplo

                        <form action="/action_page.php">
        <fieldset>
            <legend>Informações Pessoais:</legend>
            <label for="fname">Primeiro nome:</label>
            <br>
            <input type="text" id="fname" name="fname" value="John"/>
            <br>
            <label for="lname">Último nome:</label><br><input type="text" id="lname" name="lname" value="Doe"/>
            <br><br>
            <input type="submit" value="Enviar"/>
        </fieldset>
     </form>

    Resultado

    Informações Pessoais:




    O Elemento <datalist>

    O elemento <datalist> especifica uma lista de opções pré-definidas para um elemento <input>.

    Os usuários verão uma lista suspensa de opções pré-definidas conforme inserem dados.

    Exemplo

                        <form action="/action_page.php">
        <input list="browsers"/>
        <datalist id="browsers">
            <option value="Edge"/>
            <option value="Firefox"/>
            <option value="Chrome"/>
            <option value="Opera"/>
            <option value="Safari"/>
        </datalist>
     </form>

    Resultado

    O Elemento <output>

    O elemento <output> representa o resultado de um cálculo (como um executado por um script).

    Exemplo

                        <form action="/action_page.php" oninput="x.value=parseInt(a.value) + parseInt(b.value)">
        <input type="range" id="a" name="a" value="50"/> + 
        <input type="number" id="b" name="b" value="50"/> = 
        <output name="x" for="a b"></output>
        <br><br>
        <input type="submit"/>
     </form>

    Resultado

    + =

    Elementos de Formulários em HTML

    Tag Descrição
    <form> Define um formulário HTML para entrada de usuário
    <input> Define um controle de entrada
    <textarea> Define um controle de entrada multiline (área de texto)
    <label> Define um rótulo para um elemento <input>
    <fieldset> Agrupa elementos relacionados em um formulário
    <legend> Define uma legenda para um elemento <fieldset>
    <select> Define uma lista suspensa
    <option> Define uma opção em uma lista suspensa
    <button> Define um botão clicável
    <datalist> Especifica uma lista de opções pré-definidas para controles de entrada
    <output> Define o resultado de um cálculo
