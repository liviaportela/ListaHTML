• HTML Canvas

    O elemento HTML <canvas> é usado para desenhar gráficos em tempo real, via JavaScript.

    O elemento <canvas> é apenas um contêiner para gráficos. Você deve usar JavaScript para realmente desenhar os gráficos.

    O Canvas possui vários métodos para desenhar caminhos, caixas, círculos, texto e adicionar imagens.

    O Canvas é suportado por todos os principais navegadores.

    Exemplo de Canvas

    Um canvas é uma área retangular em uma página HTML. Por padrão, um canvas não tem borda nem conteúdo.

    O código para um canvas é assim:

                        <canvas id="myCanvas"width="200"height="100"></canvas>

    Nota: Sempre especifique um atributo id para ser referido em um script, e atributos width e height para definir o tamanho do canvas. Para adicionar uma borda, use o atributo style.

    Exemplo de um Canvas Básico

    Desenhar uma Linha no Canvas

    Depois de criar a área do canvas retangular, você deve adicionar um código JavaScript para fazer o desenho.

    Aqui está um exemplo de como desenhar uma linha:

                        <canvas id="myCanvas2" width="200" height="100" style="border: 1px solid #000000;"></canvas>

     <script>
        var c = document.getElementById("myCanvas2");
        var ctx = c.getContext("2d");  
        ctx.moveTo(0, 0);  // Define o ponto de início da linha
        ctx.lineTo(200, 100);  // Define o ponto final da linha
        ctx.stroke();  // Desenha a linha
     </script>

    Desenhar um Círculo no Canvas

    Este exemplo mostra como desenhar um círculo no canvas usando JavaScript.

                        <canvas id="myCanvas3" width="200" height="100" style="border: 1px solid #000000;"></canvas>

     <script>
        var c = document.getElementById("myCanvas3");
        var ctx = c.getContext("2d");
        ctx.beginPath();  // Inicia o caminho para desenhar
        ctx.arc(95, 50, 40, 0, 2 * Math.PI);  // Desenha um círculo
        ctx.stroke();  // Desenha a borda do círculo
     </script>

    Desenhar Texto no Canvas

    Este exemplo mostra como desenhar texto no canvas usando JavaScript.

                        <canvas id="myCanvas4" width="200" height="100" style="border: 1px solid #000000;"></canvas>

     <script>
        var c = document.getElementById("myCanvas4");
        var ctx = c.getContext("2d");
        // Define a fonte e o tamanho do texto
        ctx.font = "30px Arial";
        // Desenha o texto "Hello World" na posição (10, 50) do canvas
        ctx.fillText("Hello World", 10, 50);
     </script>

    Gradiente Linear no Canvas

    Este exemplo mostra como desenhar um retângulo com gradiente linear no canvas usando JavaScript.

                        <canvas id="myCanvas6" width="200" height="100" style="border: 1px solid #000000;"></canvas>

     <script>
        var c = document.getElementById("myCanvas6");
        var ctx = c.getContext("2d");
        // Cria um gradiente linear
        var grd = ctx.createLinearGradient(0, 0, 200, 0);
        grd.addColorStop(0, "red");  // Cor inicial do gradiente
        grd.addColorStop(1, "white");  // Cor final do gradiente
        // Define o estilo de preenchimento como o gradiente criado
        ctx.fillStyle = grd;
        // Desenha um retângulo preenchido com o gradiente
        ctx.fillRect(10, 10, 150, 80);
     </script>

    Gradiente Circular no Canvas

    Este exemplo mostra como desenhar um retângulo com gradiente circular no canvas usando JavaScript. 

                        <canvas id="myCanvas7" width="200" height="100" style="border: 1px solid #000000;"></canvas>

     <script>
        var c = document.getElementById("myCanvas7");
        var ctx = c.getContext("2d");
        // Cria um gradiente circular
        var grd = ctx.createRadialGradient(75, 50, 5, 90, 60, 100);
        grd.addColorStop(0, "red");   // Cor do centro do gradiente
        grd.addColorStop(1, "white"); // Cor da borda do gradiente
        // Define o estilo de preenchimento como o gradiente criado
        ctx.fillStyle = grd;
        // Desenha um retângulo preenchido com o gradiente circular
        ctx.fillRect(10, 10, 150, 80);
     </script>
