• Atributo Class em HTML

    O atributo class é usado para especificar uma classe para um elemento HTML.

    Classes são usadas para aplicar estilos a elementos específicos ou para organizar elementos semelhantes.

    Vários elementos HTML podem compartilhar a mesma classe.

    O atributo class é frequentemente usado para apontar para um nome de classe em uma folha de estilos (CSS). Também pode ser usado por um JavaScript para acessar e manipular elementos com um nome de classe específico.

    Exemplo de Uso do Atributo class

    No exemplo a seguir, temos três elementos <div> com um atributo class com o valor "city". Todos os três elementos <div> serão estilizados igualmente de acordo com a definição de estilo .city na seção <head>:

    Exemplo

                        <!DOCTYPE html>
     <html>
     <head>
     <style>
        .city {
        background-color: tomato;
        color: white;
        border: 2px solid black;
        margin: 20px;
        padding: 20px;
        }
     </style>
     </head>
    
    <body>
        <div class="city">
            <h2>Londres</h2>
            <p>Londres é a capital da Inglaterra.</p>
        </div>
        
        <div class="city">
            <h2>Paris</h2>
            <p>Paris é a capital da França.</p>
        </div>
        
        <div class="city">
            <h2>Tóquio</h2>
            <p>Tóquio é a capital do Japão.</p>
        </div>
     </body>
     </html>

    Resultado

    Londres

    Londres é a capital da Inglaterra.

    Paris

    Paris é a capital da França.

    Tóquio

    Tóquio é a capital do Japão.

    O atributo class em diferentes elementos HTML

    No exemplo a seguir, temos dois elementos <span> com um atributo class com o valor "note". Ambos os elementos <span> serão estilizados igualmente de acordo com a definição de estilo .note na seção <head>:

    Exemplo

                        <!DOCTYPE html>
     <html>
     <head>
     <style>
     .note {
        font-size: 120%;
        color: red;
     }
     </style>
     </head>
    
     <body>
        <h1>Meu <span class="note">Importante</span> Título</h1>
        <p>Este é um texto <span class="note">importante</span>.</p>
     </body>
     </html>

    Resultado

    Meu Importante Título

    Este é um texto importante.

    O Atributo class com CSS

    Para criar uma classe no CSS, escreva um caractere de ponto (.) seguido por um nome de classe. Depois, defina as propriedades CSS entre chaves { }:

    Criação de Classe

    Exemplo

                        <!DOCTYPE html>
     <html>
     <head>
     <style>
     .city {
        background-color: tomato;
        color: white;
        padding: 10px;
     }
     </style>
     </head>

     <body>
        <h2 class="city">Londres</h2>
        <p>Londres é a capital da Inglaterra.</p>

        <h2 class="city">Paris</h2>
        <p>Paris é a capital da França.</p>

        <h2 class="city">Tóquio</h2>
        <p>Tóquio é a capital do Japão.</p>
     </body>
     </html>

    Resultado

    Londres

    Londres é a capital da Inglaterra.

    Paris

    Paris é a capital da França.

    Tóquio

    Tóquio é a capital do Japão.

    Usando Múltiplas Classes

    Elementos HTML podem pertencer a mais de uma classe.

    Para definir várias classes, separe os nomes das classes com um espaço, como <div class="city main">. O elemento será estilizado de acordo com todas as classes especificadas.

    Exemplo

                        <style>
        .city {
            background-color: tomato;
            color: white;
            padding: 10px;
        }
        .main {
            text-align: center;
        }
     </style>

     <h2 class="city main">Londres</h2>
     <h2 class="city">Paris</h2>
     <h2 class="city">Tóquio</h2>

    Resultado

    Londres

    Paris

    Tóquio

    Elementos Diferentes Compartilhando a Mesma Classe

    Diferentes elementos HTML podem apontar para o mesmo nome de classe.

    No exemplo a seguir, tanto o elemento <h2> quanto o elemento <p> apontam para a classe "city" e compartilham o mesmo estilo:

    Elementos Diferentes com a Mesma Classe

    Exemplo

                        <style>
        .city {
            background-color: tomato;
            color: white;
            padding: 10px;
        }
    </style>
                    
     <h2 class="city">Paris</h2>
     <p class="city">Paris é a capital da França.</p>

    Resultado

    Paris

    Paris é a capital da França.

    Usando o Atributo class com JavaScript

    O nome da classe também pode ser usado por JavaScript para executar certas tarefas para elementos específicos.

    JavaScript pode acessar elementos com um nome de classe específico usando o método getElementsByClassName():

    JavaScript com Class

    Exemplo

                        <button onclick="myFunction()">Ocultar elementos</button>
     <h2 class="city">Londres</h2>
     <p>Londres é a capital da Inglaterra.</p>

     <h2 class="city">Paris</h2>
     <p>Paris é a capital da França.</p>

     <h2 class="city">Tóquio</h2>
     <p>Tóquio é a capital do Japão.</p>

     <script>
        function myFunction() {
            var x = document.getElementsByClassName("city");
            for (var i = 0; i < x.length; i++) {
            x[i].style.display = "none";
            }
        }
     </script>

    Resultado

    Clique em um botão para ocultar todos os elementos com o nome de classe "city":

    Londres

    Londres é a capital da Inglaterra.

    Paris

    Paris é a capital da França.

    Tóquio

    Tóquio é a capital do Japão.
