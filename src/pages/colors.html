• Cores no HTML

    As cores em HTML podem ser especificadas usando nomes de cores predefinidos, ou valores RGB, HEX, HSL, RGBA ou HSLA.

    Nomes de Cores

    Em HTML, uma cor pode ser especificada usando um nome de cor:

    Exemplo

                        <div style="background-color: Tomato; height: 25px; width: 230px;">Tomato</div>
     <div style="background-color: Orange; height: 25px; width: 230px;">Orange</div>
     <div style="background-color: DodgerBlue; height: 25px; width: 230px;">DodgerBlue</div>
     <div style="background-color: MediumSeaGreen; height: 25px; width: 230px;">MediumSeaGreen</div>
     <div style="background-color: Gray; height: 25px; width: 230px;">Gray</div>
     <div style="background-color: SlateBlue; height: 25px; width: 230px;">SlateBlue</div>
     <div style="background-color: Violet; height: 25px; width: 230px;">Violet</div>
     <div style="background-color: LightGray; height: 25px; width: 230px;">LightGray</div>

    Resultado

    Tomato
    Orange
    DodgerBlue
    MediumSeaGreen
    Gray
    SlateBlue
    Violet
    LightGray

    O HTML suporta 140 nomes de cores padrão.

    Cor das Bordas

    Você pode definir a cor das bordas:

    Exemplo

                        <h1 style="border:2px solid Tomato;">Olá, Mundo</h1>
     <h1 style="border:2px solid DodgerBlue;">Olá, Mundo</h1>
     <h1 style="border:2px solid Violet;">Olá, Mundo</h1>

    Resultado

    Olá, Mundo

    Olá, Mundo

    Olá, Mundo

    Valores de Cores

    No HTML, as cores também podem ser especificadas usando valores RGB, valores HEX, valores HSL, valores RGBA e valores HSLA.

    Exemplo

                        <h1 style="background-color:rgb(255, 99, 71);">RGB(255, 99, 71)</h1>
     <h1 style="background-color:#ff6347;">#ff6347</h1>
     <h1 style="background-color:hsl(9, 100%, 64%);">HSL(9, 100%, 64%)</h1>

    Resultado

    RGB(255, 99, 71)

    #ff6347

    HSL(9, 100%, 64%)

    Os dois elementos <div> a seguir têm sua cor de fundo definida com valores RGBA e HSLA, que adicionam um canal Alpha à cor (aqui temos 50% de transparência):

    Exemplo

                        <h1 style="background-color:rgba(255, 99, 71, 0.5);">RGBA(255, 99, 71, 0.5)</h1>
     <h1 style="background-color:hsla(9, 100%, 64%, 0.5);">HSLA(9, 100%, 64%, 0.5)</h1>

    Resultado

    RGBA(255, 99, 71, 0.5)

    HSLA(9, 100%, 64%, 0.5)
