• Comentários em HTML

    Comentários em HTML não são exibidos pelo navegador, mas podem ajudar a documentar o código-fonte HTML.

    Tag de Comentário HTML

    Você pode adicionar comentários ao seu código-fonte HTML usando a seguinte sintaxe:

                        <!-- Escreva seus comentários aqui -->

    Note que há um ponto de exclamação (!) na tag de início, mas não na tag de fechamento.

    Nota: Comentários não são exibidos pelo navegador, mas podem ajudar a documentar o código-fonte HTML.

    Adicionar Comentários

    Com comentários, você pode adicionar notificações e lembretes ao seu código HTML:

    Exemplo

                        <!-- Este é um comentário -->
     <p>Este é um parágrafo.</p>
     <!-- Lembre-se de adicionar mais informações aqui -->

    Resultado

    Este é um parágrafo.

    Ocultar Conteúdo

    Comentários podem ser usados para ocultar conteúdo. Isso pode ser útil para ocultar conteúdo temporariamente:

    Exemplo

                        <p>Este é um parágrafo.</p>
     <!-- <p>Este é outro parágrafo.</p> -->
     <p>Este é um parágrafo também.</p>

    Resultado

    Este é um parágrafo.

    Este é um parágrafo também.

    Você também pode ocultar mais de uma linha. Tudo entre <!-- e --> será oculto da exibição.

    Exemplo

                        <p>Este é um parágrafo.</p>
     <!--
     <p>Veja esta imagem interessante:</p>
     <img border="0" src="pic_trulli.jpg" alt="Trulli">
     -->
     <p>Este é um parágrafo também.</p>

    Resultado

    Este é um parágrafo.

    Este é um parágrafo também.

    Os comentários também são ótimos para depurar HTML, porque você pode comentar linhas de código HTML, uma de cada vez, para procurar erros.

    Ocultar Conteúdo Inline

    Comentários podem ser usados para ocultar partes no meio do código HTML.

    Exemplo

                        <p>Este <!-- texto interessante --> é um parágrafo.</p>

    Resultado

    Este é um parágrafo.
