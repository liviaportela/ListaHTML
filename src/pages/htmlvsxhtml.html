• HTML vs XHTML

    O que é XHTML?

    XHTML significa EXtensible HyperText Markup Language

    XHTML é uma versão mais rigorosa e baseada em XML do HTML

    XHTML é suportado por todos os principais navegadores

    Requisitos do XHTML

    • DOCTYPE é obrigatório
    • O atributo xmlns em <html> é obrigatório
    • Os elementos <html>, <head>, <title>, e <body> são obrigatórios
    • Os elementos devem estar sempre aninhados corretamente
    • Os elementos devem sempre ser fechados
    • Os elementos devem estar sempre em minúsculas
    • Os nomes de atributos devem estar sempre em minúsculas
    • Os valores de atributo devem sempre ser citados
    • A minimização de atributos é proibida

    Exemplo de Documento XHTML Mínimo

    A seguir, um documento XHTML com um mínimo de tags necessárias:

                        <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN"
     "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
     <html xmlns="http://www.w3.org/1999/xhtml">
        <head>
            <title>Título do documento</title>
        </head>
    
        <body>
            algum conteúdo aqui...
        </body>
     </html>

    Diferenças Importantes

    Os elementos do XHTML devem ser aninhados e fechados corretamente, como mostrado abaixo:

    Correto: 
                        <p><b><i>Algum texto</i></b></p>

    Resultado

    Algum texto

    Errado: 
                        <p><b><i>Algum texto</b></i></p>

    Resultado

    Algum texto

    Exemplo de elementos vazios fechados:

    Correto: 
                        <p><br /></p>
     <p><hr /></p>
     <p><img src="../media/Dog.png" alt="Descrição da imagem" /></p>

    Resultado


    Errado: 
                        <p><br></p>
     <p><hr></p>
     <p><img src="../media/Dog.png" alt="Descrição da imagem"></p>

    Resultado


    Exemplo de valores de atributos citados:

    Correto: 
                        <a href="https://www.w3schools.com/html/">Visite nosso tutorial de HTML</a>

    Resultado

    Visite nosso tutorial de HTML

    <a href="https://www.w3schools.com/html/">Visite nosso tutorial de HTML</a>

    Errado: 
                        <a href=https://www.w3schools.com/html/>Visite nosso tutorial de HTML</a>

    Resultado

    Visite nosso tutorial de HTML

    Proibição de minimização de atributos:

    Exemplo

    Correto: 
                        <p><input type="checkbox" checked="checked" /></p>
     <input type="text" name="lastname" disabled="disabled" />

    Resultado

    Errado: 
                        <p><input type="checkbox" checked /></p>
     <input type="text" name="lastname" disabled />

    Resultado
