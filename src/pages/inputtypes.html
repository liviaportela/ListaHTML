• Tipos de Elemento <input>

    Os tipos de entrada controlam o tipo de dados que o campo de entrada pode aceitar.

    Os tipos de entrada do elemento <input> no HTML são diversos. Vamos explorar alguns dos tipos mais comuns:

    Tipo de Entrada
    <input type="button">
    <input type="checkbox">
    <input type="color">
    <input type="date">
    <input type="datetime-local">
    <input type="email">
    <input type="file">
    <input type="hidden">
    <input type="image">
    <input type="month">
    <input type="number">
    <input type="password">
    <input type="radio">
    <input type="range">
    <input type="reset">
    <input type="search">
    <input type="submit">
    <input type="tel">
    <input type="text">
    <input type="time">
    <input type="url">
    <input type="week">

    Dica: O valor padrão para o atributo "type" é "text".

    Tipo de Entrada Texto

    <input type="text"> define um campo de entrada de linha única para texto:

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname"/><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname"/>
     </form>

    Resultado




    Tipo de Entrada Senha

    <input type="password"> define um campo de entrada para senha, onde os caracteres são mascarados:

    Exemplo

                        <form>
        <label for="username">Nome de usuário:</label><br>
        <input type="text" id="username" name="username"/><br>
        <label for="pwd">Senha:</label><br>
        <input type="password" id="pwd" name="pwd"/>
     </form>

    Resultado




    Tipo de Entrada Enviar

    <input type="submit"> define um botão para enviar dados do formulário a um manipulador de formulários:

    Exemplo

                        <form action="/action_page.php">
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" value="John"/><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname" value="Doe"/><br><br>
        <input type="submit" value="Enviar"/>
     </form>

    Resultado






    Tipo de Entrada Redefinir

    <input type="reset"> define um botão para redefinir todos os valores do formulário para seus valores padrão:

    Exemplo

                        <form action="/action_page.php">
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname" value="John"/><br>
        <label for="lname">Último nome:</label><br>
        <input type="text" id="lname" name="lname" value="Doe"/><br><br>
        <input type="submit" value="Enviar"/>
        <input type="reset" value="Redefinir"/>
     </form>

    Resultado






    Tipo de Entrada Rádio

    <input type="radio"> define um botão de rádio para permitir ao usuário selecionar uma opção entre várias:

    Escolha seu idioma favorito para a Web:

    Exemplo

                        <form>
        <input type="radio" id="html" name="fav_language" value="HTML"/>
        <label for="html">HTML</label><br>
        <input type="radio" id="css" name="fav_language" value="CSS"/>
        <label for="css">CSS</label><br>
        <input type="radio" id="javascript" name="fav_language" value="JavaScript"/>
        <label for="javascript">JavaScript</label>
     </form>

    Resultado



    Tipo de Entrada Caixa de Seleção

    <input type="checkbox"> define uma caixa de seleção que permite ao usuário selecionar uma ou mais opções entre várias:

    Exemplo

                        <form>
        <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike"/>
        <label for="vehicle1">Eu tenho uma bicicleta</label><br>
        <input type="checkbox" id="vehicle2" name="vehicle2" value="Car"/>
        <label for="vehicle2">Eu tenho um carro</label><br>
        <input type="checkbox" id="vehicle3" name="vehicle3" value="Boat"/>
        <label for="vehicle3">Eu tenho um barco</label>
     </form>

    Resultado



    Tipo de Entrada Botão

    <input type="button"> define um botão clicável:

    Exemplo

                        <input type="button" onclick="alert('Olá Mundo!')" value="Clique Aqui"/>

    Resultado

    Tipo de Entrada Cor

    <input type="color"> é usado para campos de entrada que devem conter uma cor.

    Exemplo

                        <form>
        <label for="favcolor">Selecione sua cor favorita:</label>
        <input type="color" id="favcolor" name="favcolor"/>
     </form>

    Resultado

    Tipo de Entrada Data

    <input type="date"> é usado para campos de entrada que devem conter uma data. Dependendo do suporte do navegador, um seletor de data pode ser exibido.

    Exemplo

                        <form>
        <label for="birthday">Data de Nascimento:</label>
        <input type="date" id="birthday" name="birthday"/>
     </form>

    Resultado

    Tipo de Entrada Data e Hora

    <input type="datetime-local"> especifica um campo de entrada de data e hora, sem fuso horário.

    Exemplo

                        <form>
        <label for="birthdaytime">Data e hora de nascimento:</label>
        <input type="datetime-local" id="birthdaytime" name="birthdaytime"/>
     </form>

    Resultado

    Tipo de Entrada E-mail

    <input type="email"> é usado para campos de entrada que devem conter um endereço de e-mail. Dependendo do suporte do navegador, o campo de e-mail pode ser automaticamente validado.

    Exemplo

                        <form>
        <label for="email">Digite seu e-mail:</label>
        <input type="email" id="email" name="email"/>
     </form>

    Resultado

    Tipo de Entrada Imagem

    <input type="image"> define uma imagem como um botão de envio.

    Exemplo

                        <form>
        <input type="image" src="../media/Dog.png" alt="Enviar" width="48" height="48"/>
     </form>

    Resultado

    Tipo de Entrada Arquivo

    <input type="file"> define um campo de seleção de arquivos com um botão "Procurar" para carregamento de arquivos.

    Exemplo

                        <form>
        <label for="myfile">Selecione um arquivo:</label>
        <input type="file" id="myfile" name="myfile"/>
     </form>

    Resultado

    Tipo de Entrada Oculta

    <input type="hidden"> define um campo de entrada oculto (não visível para um usuário). Um campo oculto permite que desenvolvedores da web incluam dados que não podem ser vistos ou modificados por usuários ao enviar um formulário.

    Exemplo

                        <form>
        <label for="fname">Primeiro nome:</label><br>
        <input type="text" id="fname" name="fname"/><br><br>
        <input type="hidden" id="custId" name="custId" value="3487"/>
        <input type="submit" value="Enviar"/>
     </form>

    Resultado




    Tipo de Entrada Mês

    <input type="month"> permite que o usuário selecione um mês e um ano.

    Exemplo

                        <form>
        <label for="bdaymonth">Data de Nascimento (mês e ano):</label>
        <input type="month" id="bdaymonth" name="bdaymonth"/>
     </form>

    Resultado

    Tipo de Entrada Número

    <input type="number"> define um campo de entrada numérico.

    Exemplo

                        <form>
        <label for="quantity">Quantidade (entre 1 e 5):</label>
        <input type="number" id="quantity" name="quantity" min="1" max="5"/>
     </form>

    Resultado

    Tipo de Entrada Intervalo

    <input type="range"> define um controle para entrada de número, como um controle deslizante (slider).

    Exemplo

                        <form>
        <label for="vol">Volume (entre 0 e 50):</label>
        <input type="range" id="vol" name="vol" min="0" max="50"/>
     </form>

    Resultado

    Tipo de Entrada Buscar

    <input type="search"> é usado para campos de busca.

    Exemplo

                        <form>
        <label for="gsearch">Buscar no Google:</label>
        <input type="search" id="gsearch" name="gsearch"/>
     </form>

    Resultado

    Tipo de Entrada Telefone

    <input type="tel"> é usado para campos de entrada que devem conter um número de telefone.

    Exemplo

                        <form>
        <label for="phone">Digite seu número de telefone:</label>
        <input type="tel" id="phone" name="phone" pattern="[0-9]{3}-[0-9]{2}-[0-9]{3}"/>
     </form>

    Resultado

    Tipo de Entrada Hora

    <input type="time"> permite que o usuário selecione uma hora (sem fuso horário).

    Exemplo

                        <form>
        <label for="appt">Selecione um horário:</label>
        <input type="time" id="appt" name="appt"/>
     </form>

    Resultado

    Tipo de Entrada URL

    <input type="url"> é usado para campos de entrada que devem conter um endereço de URL.

    Exemplo

                        <form>
        <label for="homepage">Adicione sua página inicial:</label>
        <input type="url" id="homepage" name="homepage"/>
     </form>

    Resultado

    Tipo de Entrada Semana

    <input type="week"> permite que o usuário selecione uma semana e um ano.

    Exemplo

                        <form>
        <label for="week">Selecione uma semana:</label>
        <input type="week" id="week" name="week"/>
     </form>

    Resultado
